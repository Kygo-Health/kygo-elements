# Internal & App-Store Links

> Inventory of every **internal** link (kygo.app pages) and **app-store / app-download** link
> across the `kygo-elements` components. Generated by auditing `customElements` source files.
> External research/citation links (PubMed, journals, manufacturer docs) are **not** listed here —
> see `affiliate-links.md` for Amazon/affiliate links.

Two host forms are used inconsistently across files: bare **`https://kygo.app`** and
**`https://www.kygo.app`**. JSON-LD `url` fields and canonical-style references tend to use
`www.`; in-page nav/footer links tend to use the bare host. Worth normalizing.
